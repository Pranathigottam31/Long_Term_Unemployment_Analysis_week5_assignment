# Long-Term Unemployment Data Visualization

## Project Overview
This project analyzes long-term unemployment data in the United States between 2005 and 2015. Using Python Pandas and Matplotlib this script processes the raw demographic data to generate three distinct visualizations Line Plot, Bar Plot, and Scatter Plot that highlight trends demographic impacts and market volatility.
